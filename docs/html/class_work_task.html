Detailed Description

Represents a work-related task, inheriting from the Task base class.

This class manages a work-related task, including its description, when-to-do date, deadline, priority, and the person to whom the task is assigned. It provides methods to display the task, save it to a file, load it from a stream, and convert it to a string format suitable for file storage.
