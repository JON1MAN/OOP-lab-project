\doxysubsection{Detailed Description}
Represents a task related to life activities, inheriting from the \doxylink{class_task}{Task} base class. 

This class manages a life-\/related task, including its description, when-\/to-\/do date, deadline, and priority. It provides methods to display the task, save it to a file, load it from a stream, and convert it to a string format suitable for file storage.
