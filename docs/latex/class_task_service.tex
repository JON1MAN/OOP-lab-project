void Task\+Service\+::run\+Application (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [inline]}}



Main loop of the To-\/\+Do List application. 

This function runs the main application loop, allowing the user to interact with the To-\/\+Do List application through a set of menu options. The user can view tasks for the current day, add new tasks, mark tasks as done, reschedule unfinished tasks, or exit the application.

The function operates in an infinite loop, presenting the user with a menu of options. The user selects an option by entering a number corresponding to the desired action\+:
\begin{DoxyItemize}
\item {\bfseries{1}}\+: Add a new task for today.
\item {\bfseries{2}}\+: Add a task with a custom date.
\item {\bfseries{3}}\+: Mark a task as completed.
\item {\bfseries{4}}\+: Reschedule unfinished tasks from today to the next day.
\item {\bfseries{5}}\+: Exit the application.
\end{DoxyItemize}

Depending on the user\textquotesingle{}s choice, the function invokes corresponding helper functions to perform the requested actions.


\begin{DoxyItemize}
\item {\bfseries{Invalid Input Handling}}\+: If the user enters an invalid choice, a message is displayed, and the menu is shown again.
\item {\bfseries{Exit}}\+: Choosing option 5 exits the loop and terminates the application.
\end{DoxyItemize}
