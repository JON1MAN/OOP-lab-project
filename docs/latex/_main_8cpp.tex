\doxysubsection{Detailed Description}
Main entry point for the task management application. 

This file contains the {\ttfamily main} function, which is the entry point of the task management application. It initializes the {\ttfamily \doxylink{class_task_service}{Task\+Service}} class and starts the application by calling its {\ttfamily run\+Application} method.
