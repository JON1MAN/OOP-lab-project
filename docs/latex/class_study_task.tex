\doxysubsection{Detailed Description}
Represents a task related to study, inheriting from the \doxylink{class_task}{Task} base class. 

This class manages all the necessary data for a study task, including subject, description, when-\/to-\/do date, deadline, and priority. It provides methods to display, save, load, and convert the task data to a string format for file storage.
